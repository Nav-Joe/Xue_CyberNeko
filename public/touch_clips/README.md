# 精选触摸音频库（默认模式）

桌宠**默认**点击时从此目录随机播放 `.wav`，不经过 TTS 推理。

## 目录结构

1. 将 `.wav` 放在本目录（或子目录，路径写在 manifest 里）。
2. 编辑 `manifest.json`：

```json
{
  "groups": [
    ["line01_a.wav", "line01_b.wav"],
    ["line02_a.wav", "line02_b.wav", "line02_c.wav"]
  ]
}
```

- 每个 **group** 对应一句（或一类）台词。
- group 内多个文件是同一句的变体；点击时先随机选 group，再随机选其中一个 wav。
- 路径相对于 `public/touch_clips/`。

## 入库说明

**`manifest.json` 与 `.wav` 应一并提交到 Git**，否则用户克隆后默认模式会无声。  
请使用合规来源导出的音频，勿提交侵权素材。

## 说明

- 启用 **音色工坊语料** 或 **第三方引擎语料** 后，将改走 TTS 预热缓存，不再使用本目录（除非切回「精选音频」开关）。
